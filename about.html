Algorithms Visualization

A project made by Mahmoud Ramadan & Abdo Mosa for the CS50 online course for 2020 provided by Harvard University. It's considered to be our final project for this wonderful course.

We tried to implement a useful project for our colleagues and the other students who like us want to learn algorithms in a better way. thus, we made it possible to see the algorithm running in front of your eyes and keep track of what's happenning step by step in an animated way.

We used pure JavaScript to implement this Web Application with some help of HTML and CSS but we didn't use any animation library or so.

You will find three modes to use this app and the most important one of them is the educational mode as it has the code of the algorithm and a brief description about it beside a log window that tell you the steps being performed.

We hope you enjoy it and find it useful ✌️
